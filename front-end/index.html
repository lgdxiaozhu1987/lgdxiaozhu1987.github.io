# 前端技术小结 🎨

本页面提供了前端相关技术情况, 包括但不限于 node.js 常用模块方法, javascript 速查, Babel, vue.js, react.js 等, 尽可能把平时工作中用到的一些技术和觉得好的参考资料放到这里.

所谓前端, 指的是与用户交互的前台部分，以往指的是运行在 PC 端，移动端等浏览器上展现给用户浏览的网页, 但随着互联网技术的发展，需要与用户交互的界面和设备越来越多, 前端的概念越来越广泛. 各类 HTML5，CSS3，前端框架的应用，跨平台响应式网页设计能够适应各种屏幕分辨率，完美的动效设计，给用户带来极高的用户体验。
